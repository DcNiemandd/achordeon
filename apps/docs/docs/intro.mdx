# Welcome to Achordeon

A-chord-eon is an app for writing and arranging your songs. This app came from my personal need. I wanted to export multiple songs in a nicely readable format with custom chords. After fighting with different text editors and chord placement I've had it enough and made mysef a working prototype, that proved it works and is indeed useful.

This documentation is a PRD for now. I tried to write down all the stuff I want and possibly some more. Use this information as a reason not to flame me. If you find an error in the documentation or a bug in the app you can create an [issue on the github](https://github.com/DcNiemandd/achordeon/issues).

This documentation will grow with the project. For now, a quick map:
