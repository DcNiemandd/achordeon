# Welcome to Achordeon

A-chord-eon is an app for writing and arranging your songs. This app came from a need. I needed to export multiple songs in a nicely readable format with custom chords. After fighting with different text editors and chord placement I had enough and made a prototype, that proofed it works and is indeed useful.

This documentation is a PRD for now. I tried to write all the stuff I want and possible some more. Use this information as a reason not to flame me. If you find an error in the documentation or a bug in the app you can create an [issue on the github](https://github.com/DcNiemandd/achordeon/issues).

This documentation will grow as the project does. For now, the quick map:
