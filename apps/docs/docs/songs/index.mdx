# Songs

This is the core part of this app. This app takes away from a user the hard part of making the sheets of lyrics and chords pretty. This approach allows you to easily change lyrics, chords, or chord position. Output is visible on the right side of this view.

:::danger
image of showcase song render
:::

## List

All your song are in the list view similar to file explorer. You can search in the songs and sort them by name, creation date, last change date, or by favorites. There is also a button to create a new song and import songs from a file.

:::note
Each song has a name as an identifier. The name is not its title.
:::

Possible actions for a song:

- rename
- delete
- duplicate
- make favorite
- download
- export
- edit

You can select multiple songs and make some actions in a bulk.

:::warning
There is an idea to use Google Drive as your cloud synchronization service. This feature might be implemented in a major version update after the release. Own cloud synchronization is not on my mind yet.
:::

## Song edit

You can open song editing by clicking on the edit option or by double-clicking on the song. Detailed guide for song editing is on the [next page](./editing.mdx).
