# Basics

This app has several main features that live in designated slots. You can use sidebar to navigate the app.

:::danger

Image: sidebar

- Songs
- Songbooks
- Stage
- Audience

:::

You can create `Songs` and annotate them with chords. Then you can create `Songbooks` with the songs and perform them on the `Stage` with your `Audience`, that can join your sessions.

:::danger

Icons / component for the nav items.

:::

You can try to explore for yourself in the app or you can learn more about each module in the documentation.

### Explorers

If you want to explore on your own, I have a small request for you. If you get stuck or you find some parts of this app unintuitive, create an [issue on the github](https://github.com/DcNiemandd/achordeon/issues). I will try to make your experience as smooth as possible.
