# Basics

This app has several main features that live in its designated slots. Use sidebar to navigate the app.

:::danger

Image: sidebar

- Songs
- Songbooks
- Stage
- Audience

:::

You can create new `Songs` and annotate them with chords. Bind your songs into `Songbooks` and perform them on your own virtual `Stage` with `Audience`, that will join your sessions. You can customize the app and rendering in the `Settings`.

For mobile view, these options are hidden in the popup menu and the bar is reserved for module actions.

:::danger

Icons / component for the nav items.

:::

You can try to explore for yourself in the app or you can learn more about each module in the documentation.
