# Stage & audience

One of the main features of this app is its usability for performing. You can be on [`Stage`](#stage), or you can be part of the [`Audience`](#audience).

:::note[Audience]
Having an audience is a feature that requires a running server (money). You can join an audience for free. Making a lobby will be a premium feature - it is free for everyone during testing and may become paid later.
:::

## Stage

This view is optimized for performing. You select a songbook and enter a performing mode. There you see only the song you're playing and navigation bar. You have these buttons:

- **Prev**: previous song
- **Summary**: list of songs in the songbook with search for opening specific song
- **Menu**: shows menu with options
  - Fullscreen: shows song in the fullscreen view, navigation bar hides and can be shown by clicking&nbsp;/&nbsp;taping on the song
  - Create an audience: creates a lobby with a PIN
  - Exit: leaves the performing mode and goes back to the songbook list
- **Next**: next song

You can swipe on the song to go back and forward instead of using the prev and next.

:::note
An empty songbook cannot be performed.
:::

:::note
An empty songbook cannot be performed.
:::

## Audience

To be part of an audience you need a PIN. You can insert it manually or scanning a QR code.

Audience view is similar to the stage view. You see the full render of the song the performer selected and the summary. You have these options:

- **Summary**: list of song in the songbook with search, read-only
- **Lobby**: info about the lobby - PIN and QR code
- **Song**: song selected by the performer, changes automatically

The lobby is one-time and ends when the performer ends it.
