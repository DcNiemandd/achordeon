# Stage & audience

One of main parts of this app is its usability for performing. You can be on [`Stage`](#stage), or you can be part of the [`Audience`](#audience).

:::danger[Audience]
Sending lyrics to the audience needs a running server and that costs money. For now it is free for testing. It is possible for this to became paid feature.
:::

## Stage

This view is optimized for performing. You select a songbook and enter a performing mode. There you see only the song you're playing and navigation bar. You have these buttons:

- **Prev**: previous song
- **Summary**: list of songs in the songbook with search for opening specific song
- **Menu**: shows menu with options
  - Fullscreen: shows song in the fullscreen view, navigation bar hides and can be shown clicking&nbsp;/&nbsp;taping on the song
  - Create an audience: creates a lobby with a PIN
  - Exit: leaves the performing mode and goes back to the songbook list
- **Next**: next song

You can swipe on the song to go backwards and forwards instead of using the prev and next.

## Audience

To be part of an audience you need to have a lobby PIN. You can insert it manually or you can scan a QR code.

Audience view is similar to the stage view. Only difference is that you cannot control the songs. You have these options:

- **Summary**: list of song in the songbook with search, read-only
- **Lobby**: info about the lobby - PIN and QR code
- **Song**: song selected by the performer, changes automatically
