# Songbooks

To filter out songs you want to play, you can make custom songbooks. The songbook is just an list of songs. There is always one default songbook named **All songs**.

<Image alt="Showcase songbook view" isDesign />

In the left panel you can see list of your songs. The list is the same as in `Songs` module but has some functionality disabled. On the right is your songbook list.

You can put songs in the songbook. With buttons you can insert selected songs to the start, to the end, above selected songbook song, or below. Second way is to drag&drop - dragging selected songs by the handle will show you, where will you be dropping them in the songbook.

You can reorder the songs. You can use buttons on the selected songs to move the song one song over, to the start or to the end. You can also use handle to drag&drop the song.

:::danger[FUTURE]
If you see this, drag&drop is not yet implemented.
:::

## Download

When downloading the songbook, you can set some options, that will affect the output PDF.

- **Title page**: title, subtitle, author
- **Summary**: show songs summary, show page numbers
- **Print**: paper size, double-sided

:::note[Print]
Printing settings will not change the aspect ratio of a song. If you don't select paper size, the size will be similar to A4 paper but respecting aspect ratio.
:::
